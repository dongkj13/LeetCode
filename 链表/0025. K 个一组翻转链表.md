# [25. K 个一组翻转链表](https://leetcode-cn.com/problems/reverse-nodes-in-k-group/)

## 描述

> 给你一个链表，每 k 个节点一组进行翻转，请你返回翻转后的链表。

## 题解

### 方法1

```c++
class Solution {
public:
    ListNode* reverseKGroup(ListNode* head, int k) {
        ListNode* ans = new ListNode(-1);
        ListNode* pre = ans;
        stack<ListNode*> s;
        while (head != NULL){
            ListNode* tmp = head;
            while (head != NULL && s.size() < k){
                s.push(head);
                head = head->next;
            }
            if (s.size() == k)
                while (!s.empty()){
                    ListNode* t = s.top();
                    s.pop();
                    pre->next = t;
                    t->next = NULL;
                    pre = t;
                }
            else
                pre->next = tmp;
        }
        return ans->next;
    }
};
```
